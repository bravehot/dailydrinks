
### Project Name
  dailydrinks

### Functions Brief
  * a list to list all the orders
  * add a new order
  * delete any order in the list
  * edit any order in the list
  * supports RWD

### How to run it ?
  1. `npm i` or `yarn` 
  2. `npm start`

### Directory
  * containers 
    * AddInfo // Add order information
    * HomePage // list all the orders

### Use Dependency
  * node-sass
  * react 16.8
